# 2048 Game (Physics-based)

A simple 2048 game built with **C# (Windows Forms)** and **OpenGL (via Tao Framework)**.

## How to Run

1. Clone this repository.
2. Open `Physics.sln` in **Visual Studio**.
3. If you see yellow warning icons under **References** in the Solution Explorer:
   * Right-click **References** -> **Add Reference...**
   * Click **Browse...** and select the three `.dll` files from the root folder:
     * `Tao.FreeGlut.dll`
     * `Tao.OpenGl.dll`
     * `Tao.Platform.Windows.dll`
4. Press **Start** to run the game.

## Tech Stack
* C# (.NET Framework)
* Windows Forms
* Tao Framework (OpenGL)
